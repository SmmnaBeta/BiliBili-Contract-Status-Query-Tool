## 📖 插件介绍

**B站直播间主播信息查询器** 是一款专为B站直播公会经纪人设计的Tampermonkey用户脚本。它能够在B站直播间页面自动查询并直观显示主播的详细信息，帮助经纪人快速识别有潜力的主播，提高招募效率。

## ✨ 主要功能

### 🔍 主播信息查询
- **自动识别**：进入直播间自动获取主播信息
- **实时显示**：在页面右上角直观展示关键数据
- **状态标识**：清晰标注主播的各项状态

### 🎯 核心指标展示
- **繁星主播标识**：快速识别是否繁星主播
- **入会状态检测**：明确显示可入会状态
- **新主播识别**：政策新主播特殊标注
- **历史入会记录**：查看历史入会情况

### 🎨 智能视觉提示
- **有利状态高亮**：粉色+脉冲效果突出有利信息
- **不利状态标识**：淡绿色显示不利因素
- **快速识别**：颜色编码让重要信息一目了然

### 💬 便捷操作
- **一键私信**：快速跳转私信页面联系主播
- **隐藏/显示**：灵活的界面控制
- **响应式设计**：适配不同屏幕尺寸

## 🚀 安装与使用

### 第一步：安装Tampermonkey
1. 打开浏览器扩展商店
2. 搜索 "Tampermonkey"
3. 点击安装扩展

### 第二步：安装脚本
1. 下载脚本 JS 文件
2. 导入Tampermonkey
3. 确认安装并启用脚本

### 第三步：使用插件
1. **登录B站账号**（确保有经纪人权限）
2. **访问任意B站直播间**
3. **插件自动运行**，在右上角显示主播信息
4. **查看关键指标**，快速判断主播质量
5. **点击私信按钮**直接联系意向主播
