# BiliBili-Contract-Status-Query-Tool
B站直播间主播签约状态查询器，用于直播公会快捷查询直播间主播是否签约的插件。需登录BiliBili公会后台。
# 浏览器安装Tampermonkey并添加B站直播间主播签约状态查询器教程

## 第一步：安装Tampermonkey扩展

1. 打开Microsoft Edge浏览器
2. 点击右上角的"..."菜单按钮
3. 选择"扩展" → "打开Microsoft Edge加载项"
4. 在搜索框中输入"Tampermonkey"
5. 找到Tampermonkey扩展，点击"获取"按钮
6. 在弹出的确认窗口中点击"添加扩展"



## 第二步：下载用户脚本文件

1. 访问下载地址：[https://cloud.komll.com/s/pNCq](https://cloud.komll.com/s/pNCq)
2. 点击"B站直播间主播签约状态查询器-V1.7.user.js"文件
3. 选择"下载"按钮保存文件到本地

## 第三步：安装用户脚本

### 方法一：直接安装（推荐）

1. 打点击篡改猴-->点击管理面板
2. 打开下载的`.user.js`文件所在文件夹
3. 直接将文件拖拽到浏览器窗口中
4. Tampermonkey会自动识别并弹出安装界面
5. 点击"安装"按钮完成安装

## 第四步：验证安装

1. 打开任意B站直播间页面：[https://live.bilibili.com/](https://live.bilibili.com/)
2. 等待页面加载完成后，右上角会出现主播签约状态信息卡片
3. 点击"隐藏"按钮可以暂时隐藏，右侧会出现"显示插件"标签

## 功能说明

安装成功后，插件会自动在B站直播间页面显示以下信息：

* 主播名称和头像
* 签约状态（已签约/未签约）
* 繁星等级
* 有效开播天数
* 主播类型和签约公会（如果已签约）

## 常见问题

### Q: 插件没有显示怎么办？

A:&#x20;

1. 确保已登录B站账号
2. 检查Tampermonkey扩展是否启用
3. 刷新直播间页面

### Q: 如何更新脚本？

A:

1. 下载新版本脚本文件
2. 拖拽到浏览器中覆盖安装
3. 或通过Tampermonkey管理面板更新

### Q: 如何卸载脚本？

A:

1. 点击Tampermonkey图标
2. 选择"管理面板"
3. 找到对应脚本，点击删除按钮

## 注意事项

* 该脚本需要已登录B站账号才能正常工作
* 部分直播间可能因权限限制无法获取完整信息
* 脚本仅用于个人学习和技术研究
